Web Development Cheatsheet

HTML

HyperText Markup Language (HTML) is used by web developers to create the skeletal structure of a webpage.

Anytime a new HTML document is created the following must be included:

  • The Doctype tag
  • Opening and closing html tags within in which all other parts of the document are contained
  • A head section to conatin the title and links to stylesheets, scripts, etc..
  • A body section to conatin all the elements of the webpage itself

Head section tags


Tag Name Description Attributes
<title> Title This is a tag used in the head section to name the webpage, this will appear in the tab in the user's browser. N/A
<link> Link This is a tag used in the head section most commonly to refer to an external stylesheet. rel, href, hreflang, media, referrerpolicy, sizes, type
<meta> Meta This is a tag used to define the metadata of an HTML. (data about data) charset, content, http-equiv, name

Body section tags


Tag Name Description Attributes
<div> Division Used as a container for HTML elements and helps to define a section of the document. Other tags are often used in it's place when creating a more semantic HTML document, such as: header, main, section, article, and footer. Global and Event attributes
<h1> Headings HTML suppoprts 6 different heading tags (h1-h6). Global attributes only
<p> Paragraph This is tag defines a paragraph, often used for the main bodies of text on a webpage. Global and Event attributes
